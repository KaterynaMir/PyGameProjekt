# PyGameProjekt

This is a learning project "Red Hood" that uses `pygame` library.
The aim of the game is to reach the grandmother's house while collecting as many flowers as possible.
Red Hood must avoid the enemies, one of which is the wolf chasing her.


To start the game:

```
  > python3 ./main.py
```

To run tests:

```
  > python3 ./test_character.py
  > python3 ./test_musicmanager.py
```


## Dependencies

- [pygame](https://www.pygame.org)
- [numpy](https://numpy.org)
- `unittest` for tests
